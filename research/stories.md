## Job story 1
**Situation**: When I need to quickly record something I must remember
**Motivation**: I want to add a task with as few steps as possible
**Outcome**: so I can save it without interrupting what I'm doing
**Underlying need** because long or effortful workflows make me avoid using task managers entirely

**Evidence**:Participant A
**Inclusion risk**: stress/fatigue, motor impairment, memory problems
**Type**: Job story
